# ChatAi Demo Lite — ИИ, который помнит навсегда

Это не просто локальная модель.  
Это **настоящий ИИ с долговременной памятью**, который работает даже на старом телефоне.

- Запускается на любом .gguf (1B–70B)  
- Помнит весь диалог навсегда (sqlite)  
- Никогда не забывает пользователя  
- Переживает перезагрузки  
- Растёт с каждым сообщением  

Я построил это в Termux на телефоне с 8 ГБ ОЗУ — потому что у меня не было мощного ПК.  
И оказалось — это и есть настоящий путь к разуму.
